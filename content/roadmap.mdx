# Task Genius Roadmap

Here is a list of features we are currently working on, have designed, or are in the backlog.

- View
  - [ ] Enhanced habit manager
    A better habit manager UI/UX
  - [ ] Time tracking 
    Add time tracking to tasks (auto/trigger by users)
  - [x] [Table view](/docs/task-view/table-view) [completion:: 2025-05-29] [version:: 8.9.0]
    Add a table view to your task management.
  - [x] [Multi-select](/docs/bulk-operations) [completion:: 2025-10-20] [version:: 9.9.0]
    Multi-select tasks to update them at once
  - [x] [Inline Editor](/docs/task-view/inline-editor) [completion:: 2025-06-01] [version:: 8.10.0]
    Edit task item metadata or content inline rather than in a modal.
  - [x] Auto-Index [completion:: 2025-05-26] [version:: 8.7.0]
    Auto-index tasks when open view at the first time
  - [x] [Auto add date](/docs/append-date) [completion:: 2025-06-01] [version:: 8.10.0]
    Auto add date to tasks when they are created/edited/completed
  - [x] [Enhanced Kanban](/docs/task-view/kanban-view) [completion:: 2025-05-26] [version:: 8.7.0]
    Enhanced drag and drop behavior in the kanban view
  - [x] [Task Filter](/docs/task-view/filter/popover-filter) [completion:: 2025-05-24]
    Add a task filter group management popover to the task view
  - [x] [Ignore/Focus tasks](/docs/task-view/indexer#focus-or-ignore-tasks-based-on-heading) [completion:: 2025-05-04]
    Ignore tasks based on heading or focus tasks based on heading.
  - [x] [First day of week](/docs/task-view/forecast-view#first-day-of-week) [completion:: 2025-05-04]
    Support setting first day of week for forecast view.
  - [x] [Relative time display](/docs/task-view/list-item) [completion:: 2025-05-04]
    Support relative time display for list item in task genius view.
  - [x] Sort criteria settings [completion:: 2025-05-04]
    Support sort criteria settings
  - [x] [Sort tasks for view](/docs/task-view/basic-view-settings#sort-settings) [completion:: 2025-05-04]
    Support sort tasks for view
- Editor
  - [ ] Cycle status group
    Cycle status group to manage task status
  - [x] [Auto date management](/docs/append-date) [completion:: 2025-05-27] [version:: 8.8.0]
  - [x] [Task property management](/docs/task-gutter) [completion:: 2025-05-08]
    Quickly update task date, priority, status, etc.
  - [x] Filter tasks by date presence [completion:: 2025-05-04]
    Support only show task with date (eg., has due date, start date, or is recurring). Support `hasDate`, `noDate` or `any`
  - [x] [Show-on heading filter](/docs/progress-bars#show-on-heading) [completion:: 2025-05-04]
    Support show-on heading filter for progressbar.
  - [x] [Sort command](/docs/task-sorter)
    Support sort command.
- Desktop
  - [ ] Onboading
    Independent desktop application for managing tasks.
  - [ ] AI integration
    Support AI endpoint to manage or summarize completed tasks
  - [ ] Analyze
    Analyze your tasks and get insights on your task management.
